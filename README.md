# 📰 News Magazine App

A React-based news application that fetches and displays latest news using API.

---

##  Features
-  Latest news articles
-  Category-based filtering (Sports, Tech, Business, etc.)
-  Fast and responsive UI
-  API-based dynamic content

---

##  Tech Stack
- React.js
- JavaScript
- CSS
- News API
